🥗 Smart Diet AI
AI-Powered Packaged Food Label Analyzer

Smart Diet AI is an AI-powered web application that analyzes packaged food labels using Computer Vision and OCR. Users can upload a food package image, and the system automatically detects the Nutrition Facts and Ingredients sections, extracts the text, analyzes nutritional values, identifies potentially harmful ingredients, and provides health warnings.

🚀 Features
📷 Upload packaged food images
🤖 YOLO-based Nutrition & Ingredients detection
📝 PaddleOCR text extraction
🍎 Nutrition information parser
⚠️ Harmful ingredient detection
📊 Nutrition summary
🥗 Health warnings
🔄 Automatic OCR fallback when detection fails
🌐 Web-based interface
